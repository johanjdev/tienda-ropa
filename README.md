# Tienda de Ropa - E-commerce con Next.js

Este es un proyecto de tienda real desarrollado con **Next.js**, **React** y **Supabase**.

## Cómo empezar

Primero, instala las dependencias (si acabas de clonar el repo):

```bash
npm install
Luego, inicia el servidor de desarrollo:

Bash
npm run dev
Abre http://localhost:3000 en tu navegador para ver el resultado.

 Tecnologías y Estructura
Framework: Next.js (App Router).

Estilos: Tailwind CSS.

Backend as a Service: Supabase (Autenticación y Base de Datos).

Base de datos: PostgreSQL.

Puedes empezar a editar el proyecto modificando src/app/page.tsx. La página se actualiza automáticamente.

 Despliegue
La forma más sencilla de desplegar esta aplicación es utilizando la Plataforma de Vercel.

Importante: No olvides configurar las variables de entorno de Supabase (NEXT_PUBLIC_SUPABASE_URL y NEXT_PUBLIC_SUPABASE_ANON_KEY) en el panel de control de Vercel para que la base de datos funcione en producción.
